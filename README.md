# Cypress Project For UI and Visual Automation Test

## Installation

**Prerequisites**:

* node v14.17.5
* npm 6.14.14

## Technologies used for this Web app

* node.js/npm
* javascript
* cypress
* @applitools/eyes-cypress
* mocha
* chai


# Running the App

 * 1.  Install Dependencies
 ````
    npm ci
````
* 2.  run test cases
 ````
    npm run test
````
* 3.  run test cases headless mode
 ````
    npm run test:headless
````
